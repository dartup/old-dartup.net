Dart shared hosting alpha.

I asked the community what service they first wanted Dartup to deliver and shared hosting won by a landslide. So to that goal i invite you to join me in making Dartup the best host for development and small projects.

Now at the beginning i focus mostly on getting a number of projects working on a single server. Luxury items like a control panel is lite later. So its mostly a you give me a git url and i will pull and host it. I also have written a dartup package that will make it easy to have the same server running on Dartup and locally.

Hello world example

import 'dart:io';
import 'package:dartup/dartup.dart' as dartup;
main() async{
  // Gets a HttpServer. We just use the default values localhost:8080 for local runs.
  HttpServer server = await dartup.bind();
  server.listen((HttpRequest req){
    req.response.write('Hello world');
    req.response.close();
  });
}
You can find a more complete example over at: https://github.com/dartup/hello

Terms of service:

Respect that is alpha. This is very early alpha so it not feature complete stable or secure. So don't expect it to be. So if you use Dartup for private project or anything even close to production it on your own risk. As its not really fit for any purpose.

Technical requirements:

As this is an early alpha your application need to make some small adaptations:
  • The code must be publicly available so that I can git clone it.
  • You should use the Dartup package to get port information.
  • The service right now starts bin/server.dart without arguments.
  • At this point there is no support for build tools.

How to sign up.

You only need to contact me with the following information:
  • The name you want for your project. name.dartup.io.
  • The git url to your project.
  • Any environmental variables you want to pass along to your application. Like database credentials.

Q&A

What databases/datastores are supported?
None yet. But you can host MongoDB for free over at mongolab.com and if you select Amazon Web Services EU (Ireland) region the database will be nice and close to Dartup alpha. If you search around you will find lots of services that offer free and or cheap database hosting on Amazon. And there is always firebase.
